# 에밀리의 탐정 놀이!

9월 스토브 인디 게임잼 에밀리 팀의 <에밀리의 탐정 놀이!>입니다.

### 개발기간

2021년 9월 24일 ~ 2021년 9월 26일 (3일)

### 개발 인원

에밀리 팀(8명)

- 기획 : 오혜빈
- 아트 : 김하늘, 움(가명), 전용현
- 프로그래밍 : 김윤성, 박형진, 유정민, 이원석

### 개발 환경

- Unity Engine (2020.3.4f1, 2D)

### 게임 소개

- 제목 : 에밀리의 탐정 놀이!

- 장르 : 추리, 공포

- 플랫폼 : 윈도우

- 조작 : 마우스

- 탐정 놀이를 좋아하는 주인공 에밀리! 마을에 발생한 살인 사건을 파헤치게 됩니다.

  - 사건 현장 근방을 직접 조사하고 단서를 수집해, 범인을 검거하는 스토리 진행 방식의 추리 공포 게임입니다!
    캐릭터들의 대화와 단서를 유심히! 집중해 살펴보고 범인을 골라, 반전 엔딩들을 확인해보세요!

    ![img](https://d2x8kymwjom7h7.cloudfront.net/live/application_no/96001/application_no/96001/default/3997e60fefd94d9f9d69f10c4571267e.png)

  - 게임 진행을 위해 필요한 것은, 오직 마우스 좌측 버튼 클릭과 드래그!
    1. 캐릭터 혹은 아이템을 누르면 대화창이 출력되고, 대화창 영역을 다시 누르면 대화창은 사라집니다!
    2. 상단 중앙에 배치된 이동 버튼을 눌러 다른 장소로 이동이 가능합니다.
    3. 특정 물건을 드래그해서 새로운 증거를 발견해보세요!
